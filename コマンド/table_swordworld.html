SW経歴表
  • 分類：ダイスロールコマンド
  • 権限：権天使以上
  • 書式：「黒、経歴表[経歴表名][振る数]」
  • 例文：
    • 「黒、経歴表」
    • 「黒、経歴表ABCD」
    • 「黒、経歴表10」
    • 「黒、経歴表S4」
  • 概要：ソード・ワールド2.0用の経歴表を振る。
    デフォルトでは経歴表名=ABC、振る数=3である。
    ・経歴表名
    • A : 経歴表A
    • B : 経歴表B
    • C : 経歴表C
    • D : 経歴表D
    • Z : 蛮族経歴表A
    • Y : 蛮族経歴表B
    • X : 蛮族経歴表C
    • G : 魔法文明時代経歴表
    • S : センティアン経歴表
